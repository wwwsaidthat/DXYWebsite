# 微观经济学名词解释

# CH1 导论

1.机会成本：为了得到某东西所放弃的最宝贵的东西

2.理性人：系统而有目的地尽最大努力实现目标的人，比较成本和利益的边际变动而做决定

3.经济人：理性（选择最有利的方案） + 自私（追求自身利益最大化）

4.激励：引起某个人做某种行为的东西，比如奖励或惩罚的预期

5.市场失灵：市场本身不能有效配置资源的情况（外部性、市场势力）

6.生产要素：经济体用来生产物品和劳务的资源

7.model : 生产可能性边界：表示在可得到的生产要素和生产技术既定时，一个经济体所能生产的两种产品的数量的各种组合的图形

8.社会面临权衡取舍：得到更多的一种物品要以减少另一种物品为代价

# CH2 需求、供给和价格机制

1.需求：在一段时期内，消费者收入和其他条件不变的情况下，特定价格下消费者愿意购买并且能够购买的该商品的数量

2.个人需求：个别消费者对某一个商品的需求
    市场需求：一个商品市场上所有消费者对所有厂商生产的某种商品的需求

3.替代效应：X价格上升 -> 相对Y价格变贵 -> 增加Y购买替代X
    收入效应：X价格上升 -> 收入购买力下降 -> 减少X又减少Y（正常商品）

4.需求量的变动：价格变动引起消费者对商品的需求数量改变（曲线不变）
   需求的变化：其他因素改变引起消费者对商品需求的改变（曲线改变）

5.替代品：一种商品价格上升另一种商品需求量增加
   互补品：一种商品价格上升另一种商品需求量减少

6.供给：特定条件下，生产者愿意供给并且能够供给的商品数量

7.厂商供给：单个厂商供给单个商品
   市场供给：所有厂商对某个产品的供给

8.供给量的变动：价格变化引起生产者对供给量的改变
   供给变动：影响供给的非价格因素改变使生产者改变供给

9.均衡：一个经济体中特定的经济单位或经济变量在一系列经济力量相互制约下达到的相对静止并保持不变的状态
   供求均衡：供给和需求相等时商品供给和价格不再变动的状态

10.支付意愿：一个消费者愿意为一种物品支付的最高价
                          衡量买者对物品的评价

11.消费者剩余：消费者愿意购买的价格减实际购买花费的价格
      消费者剩余 = 需求曲线以下价格以上围成的图形面积

12.成本：生产者生产某种商品必须消耗的每种东西的价值

13.生产者剩余：生产者卖出的价格减商品的生产成本
      生产者剩余 = 价格以下和供给曲线以上围成的面积

14.社会总福利 = CS + PS = 参与市场贸易得到的总收益 = 买者的评价- 卖者的成本

15.弹性：衡量一个经济量对另一个经济量变动的敏感程度的指标

16.需求价格弹性：一种商品需求量的变动对于价格的变动的反应程度
      拇指规则：过某一点的需求曲线越平坦，需求价格弹性越大，过某一点的需求曲线越陡峭，需求价格弹性越小

17.需求收入弹性：一定时期内，消费者对某种商品需求量的变动对收入变动的反应程度
                                  需求量对价格变化的敏感性

18.需求交叉（价格）弹性：一定时期内，一种商品需求量对于相应相关的一种商品价格相对变动，一相对变动的敏感程度
                                                   替代效应 E > 0 ， 收入效应 E < 0

19.供给弹性：一定时期内，相应于商品价格的相对变动，一种商品供给量相对变动的敏感程度

20.社会净损失：

21.社会福利净损失：
      缺乏弹性对应的是社会福利净损失小

# CH3 消费者选择

1.欲望    有限性：对某种商品的欲望是有限的
                无限性：一种欲望满足后，更高层次的欲望就会产生

2.效用：消费者购买某种商品或服务获得的满足感，是人的主观感受
    基数效用：以基数个单位的效用表示消费者消费一定量的商品或服务所获得的效用
    序数效用：给不同商品按效用满足高低进行排序

3.总效用：既定数量消费中获得的总满足程度
    边际效用：消费者每增加消费一单位商品或服务所增加的效用

4.边际效用递减规律：在一定时期内，随着消费者不断增加某种商品或服务的消费量，在其他商品或服务消费量不变的条件下，每增加一单位商品或服务消费获得的效用增加量递减

5.两种商品时消费者的选择行为：在收入既定的条件下，合理配置两种商品的消费数量

6.需求曲线的推导：在其他条件不变时，对应既定的价格，消费者选择效用最大的商品数量组合，这一数量就是该价格下的需求量，即消费者的需求曲线来源于消费者效用最大化的均衡条件
   为什么需求曲线向右下方倾斜？
               因为遵循 MU / P = λ ， 当价格升高，减少购买量会获得更高的边际效用
               两种商品时同样满足上式，当A产品价格上升时，根据替代效应，减少A购买增加B购买

7.偏好的性质：不饱和性、多样性、可排序性、传递性

8.无差异曲线：在既定偏好条件下，消费者可选择的带来相同满足程度的两种商品的数量组合连成的图形

9.边际替代率（MRS）：在效用既定时，增加一单位某商品的消费可代替的另一商品的消费数量

10.边际替代率递减：在效用既定时，随着一种商品消费数量增加，增加一单位该商品消费可代替的另一商品的消费量逐渐减少

11.预算约束线：在收入和商品价格既定时，全部收入所能购买的各种商品的商品数量组合

12.消费者均衡：预算约束线与无差异曲线相切的切点

13.收入——消费扩展线：不同收入水平的预算约束线，消费者分别选择效用最大化的消费数量组合，这些均衡点描绘出的一条曲线

14.价格——消费扩展线：一种商品价格改变，收入与其他商品价格不变时，消费者均衡点的轨迹

15.恩格尔曲线：某种商品消费数量随收入变动而变动的轨迹

16.恩格尔定律：家庭食物支出在总支出中所占比重随家庭收入增加而递减

17.从价格扩展线到需求曲线：借助价格——消费扩展线可以得出一系列可能价格与相应需求量之间的关系，从而得到消费者的需求曲线

18.在效用最大化的驱动下，价格变化会通过收入效应和替代效应促使消费者改变对商品的消费量

19.替代效应：一个商品价格变化会改变相对价格，鼓励消费者用更便宜的商品去替代相对昂贵的商品
      收入效应：一个商品价格改变归影响收入购买力，促使消费者购买更多会更少商品以保证效用最大化

20.正常商品和普通低劣品：替代效应 > 收入效应
     吉芬商品：替代效应 < 收入效应

# CH4 企业的生产和成本

1.企业的目标：利益最大化

2.生产函数：技术水平不变，企业在一定时期内使用各种生产要素的数量与所能生产的最大产量的关系

3.固定比例的生产函数：生产要素按固定比例搭配，产量取决于较少的生产要素
   线性生产要素：生产要素可以相互替代
   柯布—道格拉斯生产函数：劳动和资本不完全替代

4.可变要素：可调整的生产要素
    不变要素/固定要素：不能或来不及调整的生产要素

5.边际报酬递减规律：技术给定时，其他条件不变，一种可变要素增加到一定程度后，所带来的边际产量递减

6.等产量线：技术水平不变的条件下，由生产相同产量所需的生产要素不同数量组合描绘出的曲线

7.边际技术替代率：产出水平不变，增加一单位生产要素投入可代替的另一种生产要素的投入量

8.边际技术替代率递减：产量不变的条件下，随着一种生产要素数量的增加，每增加一单位该生产要素所能替代的另一生产要素的数量递减
    内在原因：边际报酬递减规律

9.等成本线：生产要素价格既定，花费相同成本可购买的两种生产要素的不同数量组合

10.生产者均衡：等成本线和等产量线切点

11.生产扩展线：生产要素价格不变和其他条件不变，随着成本的增加，生产者均衡点连成的曲线

12.生产成本：一定时期内，企业生产一定数量的产品所使用的生产要素的费用
      会计成本（显性成本）：会计账户上记录的企业在过去一段时期内生产和经营的实际支出
      机会成本（经济成本）：某项资源用于特定用途而不得不放弃其他机会所带来的成本

13.会计利润：企业销售产品所获的收益与会计成本之间的差额
      经济利润：企业销售产品所获的利益与经济成本之间的差额
      正常利润：企业家自有资源的机会成本

14.可回收成本：某些成本产生后可以通过出售或出租方式很大程度加以回收
      沉没成本：已发生而无法回收的成本
                           沉没成本的机会成本为0，不影响短期决策

15.不变成本：不随企业产量变动而变动的那部分成本
      可变成本：随企业产量变动而变动的成本

16.为什么SMC曲线成U形？边际报酬递减规律
      为什么LMC曲线成U形？规模报酬的变化

17.规模经济：企业扩张时，产量扩大一倍，生产成本扩大小于一倍
                         长期平均成本下降
      原因：专业化，设备的不可分性，范围经济，节约管理费用，融资、购货、销售优势

18.规模不经济：企业产量扩大一倍，生产成本扩大不止一倍
      原因：管理和协调不灵活，应变能力变弱，决策效率低且信息成本高

19.规模变化：所有投入要素同比例变化
     
20.规模报酬：所有投入要素变化引起的产出变化
      规模报酬变化：规模报酬递增 -> 规模报酬不变 -> 规模报酬递减

21.规模报酬和规模经济：规模报酬递增一定规模经济，规模经济不一定对应规模报酬递增

# CH5 完全竞争市场

1.市场：将买卖各方联系在一起的纽带

2.划分依据：卖者数量、相对规模、集中度
                        产品同质/差异
                        进入壁垒
                        完全/不完全信息，对称/非对称信息

3.四种代表性市场类型：完全竞争市场
                                             不完全竞争市场：垄断竞争、寡头竞争、完全垄断

4.完全竞争市场的特点：无数厂家、同质化产品、没有进出壁垒、完全信息
                                             --> 价格接受者

5.完全竞争市场的收益： R = PQ = P * Q
                                             平均收益AR = 边际收益MR = P *

6.厂商利润最大化：MR = MC 且 MC 处于上升区段

#### 以下为完全竞争市场

7.短期供给决策：P > ATC : 盈利
                                P = ATC：不盈不亏
                                AVC < P < ATC：亏损但持续经营
                                P < AVC：亏损且立即停业

   长期供给决策：P > ATC : 盈利，获得超额利润
                                P = ATC：不盈不亏（零利润均衡）
                                P < ATC：厂商缩小规模直到退出市场

8.完全竞争企业短期供给曲线：MC 曲线在 AVC 曲线之上的那一部分

9.生产者剩余：与消费者剩余一样，除了最后一单位产出，对于每个单位产量，生产者均可获得生产者剩余
                            是所有生产单位的边际成本与市场价格之间的差额的总和

10.生产者剩余 PS = R - VC
     利润 Π = R - VC - FC

11.零利润均衡法则：P = MC = AC
      零利润：企业家凭自有才能获得应得的收益

12.短期均衡（短期利润最大化）：P* = MR* = MC* = AR* = AC*
     长期均衡（长期利润最大化）：P* = LMR* = LMC = LAR* = LAC    

13.如果要素市场存在“外部经济”，当要素价格随着新企业的进入（或原有企业的退出）不升反降（或不降反升）时，长期供给曲线向右下方倾斜
      买多了反而便宜

14.蛛网模型：收敛型蛛网（需求弹性 > 供给弹性）
                          发散型蛛网（需求弹性 < 供给弹性）
                          循环型蛛网（需求弹性 = 供给弹性）

15.蛛网模型的形成原因：需求具有当期性，生产（供给）具有滞后性

# CH6 不完全竞争市场

1.垄断的特征：一个卖家和众多的买家，一种商品，存在进入壁垒

2.垄断的类型：特许垄断、专利垄断、自然垄断、资源垄断

3.垄断企业的利益最大化：MC = MR
   垄断企业不能随心所欲制定价格，需要了解市场需求的特征（厂商的约束：需求曲线）
   垄断企业在边际成本等于边际收益的产量上实现利润最大化

4.垄断企业不存在短期供给曲线：供给曲线反应价格和供给量的一一对应关系，垄断条件下价格和供给量并无一一对应关系
   原因：垄断企业面对需求曲线，需求曲线向右下方倾斜，边际收益曲线不同于需求曲线

5.垄断企业的价格和供给量并不一一对应
   垄断企业的MR与企业无关，与市场有关，定价只与需求曲线有关

6.垄断企业短期均衡与完全竞争市场相同，长期均衡只有盈利/不盈不亏两种

7.垄断条件下，行业规模调整只有退出没有进入

8.自然垄断：不变成本非常大，可变成本和边际成本非常小，随产量增加平均成本不断下降，产量到充分大时才到达最低点

9.自然垄断企业的形成和维持：
   形成：最有效率的企业通过价格战，把其他企业全部排挤出去，直到最后独占整个市场，形成垄断
   维持：形成垄断后，企业可凭借垄断地位按照利润最大化的要求重新决定自己的产量和价格

10.价格歧视：
      一级价格歧视：一人一价，将每单位产品价格定在消费者意愿支付水平上
      二级价格歧视：按量其实，根据购买量制定不同价格
      三级价格歧视：一群一价，对不同需求价格弹性的消费群体收取不同价格
                                 （弹性高：低价，弹性低：高价）

11.市场势力的测度：P - MC  又有  ( P - MC ) / P = 1 / E
      垄断：P > MC = MR
      竞争：P = MC = MR

12.垄断竞争：每个企业在整个市场中所占份额小，生产差异化产品

13.垄断企业的长期均衡：零利润均衡

14.寡头垄断：少数几家企业掌握大部分或全部市场，相互依赖和影响
     市场容量有限和规模经济对生产规模的要求 -> 市场上只能存在少量企业

15.寡头垄断的特点：策略性互动
     存在一个连续不断的“影响—反应链条”，寡头企业最初行动的结果如何，取决于其他企业的反映情况
     每个企业的行为举足轻重，实质性影响到其他企业
     企业必须考虑其他企业的反应

16.古诺模型的两个假设：
      其他企业的产量是既定不变的
      生产同质化产品且成本为0

17.斯塔尔博格模型的假设：
     某一厂商先定产量
     生产同质化产品成本为0
     后定产量的厂家产量根据先定产量的厂家的产量再做决定

​     先定产量者与古诺模型存在区别，必须考虑后定产量厂家的反应

18.先行者收益：在斯塔尔博格模型中，先行者决定的产量为垄断时的产量

19.伯川德模型：寡头垄断市场通过价格来竞争
      最终结果：P = MC = 0

​      纳什均衡就是完全竞争的均衡

20.斯威齐模型：解释寡头市场价格稳定性
      寡头企业改变价格，其他企业反应不对称 -> 寡头企业不会轻易调整价格

​      存在价格反转点和价格刚性 
​       -> 成本改变时，短期来看寡头企业可能不会调整价格和产量

21.卡特尔：寡头企业联合成一个垄断企业，追求总利润最大化
      特点：不稳定性（因为所有成员都有增产的趋势）

22.完全竞争市场：静态效率高，动态效率低

23.从静态角度看（技术不变），完全竞争企业是有效率的，不完全竞争企业是没有效率的
     对于完全竞争企业，不完全竞争企业的技术进步可能更快一些，这会抵消掉静态的低效率还有余，从而是他综合效率超过完全竞争企业

# CH7 市场失灵和微观经济政策

1.市场失灵：市场机制不能有效发挥作用，难以实现帕累托效率

2.租金：支付给生产要素的收入超过提供这个要素价格的最低价格之间的差值

3.寻租：搜寻低于可以用低于垄断企业经济利润的价格购买垄断的权力，寻求获得现有的垄断权力
               行为不创造价值但拿走他人价值

4.外部性：一个个体的经济活动给其他人带来未经过市场过程的影响

5.干预企业外部性的基本要求：企业的边际成本和边际收益同其相应的边际社会成本和边际社会收益相等

6.科斯定理：交易成本足够低时，只要产权界定清楚，不论最初如何界定，私人之间在自利驱动下会选择协商和交易，一举解决外部性问题（利用市场解决一切问题）

7.交易成本：为了达成交易而付出的各种费用
